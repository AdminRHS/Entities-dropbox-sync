## TASK ID NAMING CONVENTION

### Format: `TASK-[DEPT]-###`

**Examples:**
- `TASK-LGN-001` = Lead Generation task #1
- `TASK-DEV-042` = Development task #42
- `TASK-HRM-015` = HR task #15
- `TASK-CONT-001` = Continuous task #1 (department-agnostic)

### Project ID Format: `PROJ-[DEPT]-###`

**Examples:**
- `PROJ-LGN-001` = Lead Generation project #1 (Website Launch)
- `PROJ-DEV-001` = Development project #1
- `PROJ-HRM-001` = HR project #1

### Milestone ID Format: `MLT-[DEPT]-###`

**Examples:**
- `MLT-LGN-001` = Lead Gen milestone #1 (Content Optimization)
- `MLT-DEV-001` = Development milestone #1
- `MLT-HRM-001` = HR milestone #1

### Step ID Format: `STT-###` (sequential across all tasks)

**Examples:**
- `STT-001` = Step #1 (any department)
- `STT-034` = Step #34 (any department)
