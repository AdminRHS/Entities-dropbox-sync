# RESEARCHES ISO Code Registry

## Overview
This document defines the ISO code system for Researches in the TASK_MANAGERS taxonomy. The code `RSR` follows the standard 3-letter consonant-only convention and does NOT end with 'T' as research sessions are non-template entities.

## ISO Code Definition

### Primary Code
- **Code**: `RSR`
- **Full Name**: Researches
- **Entity Type**: Research Sessions and Documentation
- **Description**: Research sessions, gap analyses, documentation, and intelligence gathering outputs
- **Template Status**: Non-template (no 'T' suffix)

### Code Structure
```
RSR-XXX
│   └── Sequential 3-digit number (001-999)
└── Researches identifier
```
