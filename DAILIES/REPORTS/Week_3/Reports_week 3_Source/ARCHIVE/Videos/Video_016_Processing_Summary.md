## ‚úÖ KEY INSIGHTS FROM VIDEO_016

### Morgan Brown's Core Principles (PM Methodologies)

1. **Learning Rate Over Activity Rate**
   - Optimize for learning per unit of effort, not just velocity
   - Compounds over time; builds institutional knowledge
   - Documented in TOOL-118 as core decision-making principle

2. **Explicit Principles for Decisions**
   - Make decisions based on documented criteria, not gut feel
   - Enables faster decisions, delegation, and stakeholder alignment
   - Implemented in TOOL-118 and Task-Template-067

3. **Systematize Recurring Workflows**
   - Invest in templates and frameworks for tasks you repeat
   - Custom prompts (TOOL-116), decision frameworks (TOOL-118)
   - ROI realized after 3-5 uses of template

4. **Multi-Tier Evaluation for Cost Optimization**
   - Use cheap models for obvious cases, expensive for borderline, humans for edge cases
   - 70-85% cost reduction vs. using expensive model for all evaluations
   - Documented in TOOL-115 and Task-Template-065

5. **Brand Consistency at Scale with AI**
   - Company Voice Guide ensures AI-generated content matches brand
   - Critical for customer-facing AI features
   - Implemented in TOOL-117 and Task-Template-064

### Dropbox Product Strategy Insights

1. **Universal Search as Core Value Proposition**
   - Dash addresses app proliferation problem (avg worker uses 10+ SaaS tools)
   - 100+ integrations, single search interface
   - ROI: 2-4 hours saved per week per user

2. **Stacks: Living Documents That Auto-Update**
   - Smart collections that monitor search criteria continuously
   - Ideal for research monitoring, competitive intel, recurring topics
   - Reduces manual aggregation from 2-4 hours/week to 30-60 min

3. **Security + Usability Integration**
   - Protect and Control provides governance layer across Dash ecosystem
   - Cross-app visibility, automated policy enforcement
   - Addresses enterprise concern about data sprawl
