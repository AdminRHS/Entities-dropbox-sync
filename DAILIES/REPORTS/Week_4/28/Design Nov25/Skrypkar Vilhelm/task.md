## Task 2: Review Generated Mascot Graphics (Quality Control)

### Steps:
1. **Technical Validation:**
   - Check file format (PNG or SVG)
   - Verify dimensions (800x800 minimum, 2000x2000 maximum, 1024x1024 acceptable)
   - Confirm file size (under 2MB)
   - Validate naming convention: `mascot_[department]_[role]_[description].png`

2. **Design Validation:**
   - Verify no forbidden objects: gadgets, laptops, displays, pens, notebooks, tombstones
   - Check bottom area is clear (will be covered by title)
   - Confirm mascot matches approved design
   - Verify versatile poses and creative environments
   - Ensure no text/titles on image (mascot + background only)

3. **Documentation Check:**
   - Confirm prompt documented in [spreadsheet](https://docs.google.com/spreadsheets/d/12s6qj8pahYQ4hKQrzCgQsu20KobpkoHVm9ax2HVeORc/edit?usp=drive_link)
   - Verify file uploaded to correct folder

4. **Feedback Delivery:**
   - If approved: Mark as complete in tracking
   - If revisions needed: Provide clear, specific feedback in Notion task comments
   - Document patterns for future reference
