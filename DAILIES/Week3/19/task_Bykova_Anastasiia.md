## Task 2: Develop Course Plan for Second Course

### Steps:
1. Open Gemini AI and prepare the completed transcript
2. Create a prompt asking Gemini to analyze the transcript and suggest course structure
3. Request 3-4 lesson breakdown with learning objectives
4. Review Gemini's suggested structure and refine as needed
5. Define clear learning outcomes for each lesson
6. Create lesson titles and descriptions
7. Document the course plan in Google Drive

### Resources and Links:
- [Gemini AI](https://gemini.google.com/)
- [First Course Reference](https://drive.google.com/drive/folders/1WMaQEhj7wV-IPDP2qAy3KWzR-8U-GoOt)
- [Game Academy Platform](https://gameacademy.com/)

### Instructions:

**Gemini Prompt Template:**
"I have a video transcript for an educational course. Please analyze this content and create a structured course plan with 3-4 lessons. For each lesson, provide:
- Lesson title
- Learning objectives (3-5 per lesson)
- Key topics covered
- Suggested activities or examples
- Estimated duration

Here is the transcript: [paste transcript]"

**Course Planning Steps:**
1. Paste transcript into Gemini with the prompt above
2. Review Gemini's output for logical flow and completeness
3. Adjust lesson boundaries if content is unbalanced
4. Add specific learning outcomes for each lesson
5. Identify any gaps that need additional content
6. Create a course overview document with:
   - Course title and description
   - Target audience
   - Prerequisites
   - Learning outcomes
   - Lesson breakdown
7. Save as "[Course-Name]-Plan.md" in Google Drive
