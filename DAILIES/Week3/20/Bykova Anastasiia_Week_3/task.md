## Task 1: Improve Black Friday Banner Designs

### Steps:
1. Open ChatGPT and prepare mascot reference images
2. Write detailed prompt describing Black Friday banner requirements:
   - Specify "3 Free Working Days" as primary message
   - Include mascot integration requirements
   - Define color scheme (brand colors + black elements for theme)
   - Specify square format for LinkedIn compatibility
   - Request multiple variations (10+ options)
3. Generate initial batch of designs using ChatGPT
4. Select best 3-5 designs for refinement
5. Import selected designs into Figma
6. Manually adjust text size for mobile readability
7. Add Black Friday thematic elements (gifts, sale tags, etc.)
8. Ensure proper visual hierarchy with "3 Free Working Days" prominent
9. Test designs on mobile viewport
10. Export final versions in high quality
11. Upload to Google Drive portfolio folder

### Resources and Links:
- [ChatGPT](https://chat.openai.com)
- [Figma workspace](https://www.figma.com)
- [Mascot reference folder](https://drive.google.com/drive/folders/1Hqf6gA1MSJ2B4p5izBDOiogW-ztghwlz?usp=drive_link)
- [Portfolio folder](https://drive.google.com/drive/folders/11_g5MfEmfNgmssCaftVMokJGwBpz0rxm)

### Instructions:

**ChatGPT Prompt Template:**
```
I need to create a Black Friday banner for LinkedIn (square format, 1080x1080px).

Requirements:
- Main message: "3 Free Working Days" (must be largest, most prominent text)
- Secondary text: "Black Friday Sale"
- Include my brand mascot (attached reference image)
- Color scheme: [Your brand colors] with black elements for Black Friday theme
- Style: Professional, clean, modern
- Add thematic elements: gift boxes, sale tags
- Text must be readable on mobile devices (minimum 24px)

Please generate 10 different layout variations. Do not add any text beyond what I specified.
```

**Figma Refinement Process:**
- Check text readability at 375px width (mobile)
- Adjust font sizes if needed (body text minimum 16px, headlines minimum 24px)
- Ensure 40:60 contrast ratio between text and background
- Add subtle shadows or backgrounds to improve text legibility
- Verify mascot doesn't overlap critical text
- Export as PNG at 2x resolution for quality

### Instructions:

**Animation Types to Practice:**
1. **Text Animation**: Title reveal, fade-in/out
2. **Element Animation**: Icons sliding in, buttons highlighting
3. **Transition Animation**: Scene changes, crossfades
4. **Motion Graphics**: Simple moving backgrounds

**Runway Workflow (10-minute process):**
1. Import static design (2 minutes)
2. Select animation type and parameters (3 minutes)
3. Preview and adjust (2 minutes)
4. Render and export (3 minutes)

**Export Settings:**
- Format: MP4 for video, GIF for web
- Resolution: 1080p minimum
- Frame rate: 30fps for smooth motion
- File size: Under 10MB for easy sharing

**Portfolio Integration:**
