**Format Adaptation Strategies:**

**Vertical Carousel (1080x1920):**
```
Composition approach:
- Character positioned in upper 2/3
- Key lesson elements visible in frame
- Text space reserved at bottom if needed
- Environment provides context above/below character

Prompt additions:
"portrait orientation, 9:16 aspect ratio, character centered
vertically with environment extending top and bottom"
```

**Vertical Cover (1080x1350):**
```
Composition approach:
- More balanced vertical composition
- Character and environment equally weighted
- Tighter framing than carousel format
- Works well for thumbnails

Prompt additions:
"portrait orientation, 4:5 aspect ratio, balanced composition,
character and scene well-framed within vertical format"
```

**Horizontal Cover (1920x1080):**
```
Composition approach:
- Character positioned left or right
- Environment spreads across frame
- Wider scene context visible
- Good for header images

Prompt additions:
"landscape orientation, 16:9 aspect ratio, character positioned
[left/right], scene extends across horizontal frame"
```

**Format Generation Workflow:**

**Option 1: Generate Each Format Separately**
```
Advantages:
- Optimal composition for each format
- Better quality control
- Format-specific adjustments possible

Process:
1. Write format-specific prompt variant
2. Generate image for specific aspect ratio
3. Review and refine
4. Repeat for each format needed
```

**Option 2: Generate Master, Then Crop/Adapt**
```
Advantages:
- Faster for multiple formats
- Consistent base image
- Less regeneration needed

Process:
1. Generate large, high-resolution master image
2. Crop or adapt to each required format
3. Adjust composition as needed for each crop
4. May require regeneration if crops don't work

Note: This approach may compromise quality - prefer Option 1
for professional course materials.
```

**Quality Check Per Format:**

‚ñ° Composition works well in format
‚ñ° No important elements cut off
‚ñ° Text space available if needed
‚ñ° Image quality maintained (no pixelation)
‚ñ° File size appropriate for platform

Test on actual platform if possible to verify appearance.
