## Task 1: Подготовка документов для новых сотрудников на понедельник

### Steps:
1. Проверить список сотрудников, которые начинают работу в понедельник
2. Подготовить все необходимые договоры согласно процедурам, изученным у Жени
3. Правильно заполнить все таблицы с информацией о новых сотрудниках
4. Проверить корректность всех данных в документах
5. Подготовить пакеты документов для каждого нового сотрудника
6. Убедиться, что все документы подписаны правильно

### Resources and Links:
- Шаблоны договоров
- Таблицы для новых сотрудников
- CRM система
- Процедуры оформления документов от Жени

### Instructions:

1. Начните с проверки списка новых сотрудников в CRM или таблице планирования
2. Для каждого сотрудника создайте договор, следуя процедурам, которые показала Женя:
   - Используйте правильные шаблоны
   - Введите все необходимые данные в соответствующие поля
   - Проверьте правильность заполнения всех разделов
3. Внесите информацию о каждом сотруднике в соответствующие таблицы:
   - Таблица новых сотрудников
   - Таблица договоров
   - Другие связанные таблицы
4. После заполнения проверьте каждую запись на наличие ошибок
5. Соберите полный пакет документов для каждого сотрудника
6. Убедитесь, что все документы готовы к выдаче в понедельник

## Task 2: Поддержание актуальности информации в CRM

### Steps:
1. Ежедневно проверять новые записи в CRM
2. Обновлять информацию о статусах кандидатов
3. Вносить новых кандидатов из джоб сайтов в CRM
4. Проверять корректность существующих данных
5. Исправлять выявленные ошибки и несоответствия
6. Регулярно синхронизировать данные между таблицами и CRM

### Resources and Links:
- CRM система
- Таблицы кандидатов
- Джоб сайты
- Контакты кандидатов

### Instructions:

1. В начале каждого рабочего дня проверяйте CRM на наличие новых записей или обновлений
2. Для каждого нового кандидата с джоб сайтов:
   - Создайте новую запись в CRM
   - Внесите всю доступную информацию (имя, контакты, позиция, статус)
   - Свяжите запись с соответствующей вакансией
3. Регулярно обновляйте статусы кандидатов (новый, собеседование, предложение, принят, отклонен)
4. Проводите проверку данных на корректность:
   - Проверьте формат контактов
   - Убедитесь в правильности названий позиций
   - Проверьте соответствие данных между таблицами и CRM
5. При обнаружении ошибок немедленно исправляйте их
6. Ведите учет всех изменений для отслеживания истории

## Task 3: Оперативная работа с кандидатами на джоб сайтах

### Steps:
1. Проверять сообщения от кандидатов несколько раз в день
2. Отвечать на все входящие сообщения в течение рабочего дня
3. Обрабатывать накопившиеся сообщения после выходных в первую очередь
4. Предоставлять полную и полезную информацию кандидатам
5. Отслеживать статусы коммуникаций с кандидатами
6. Обновлять информацию о взаимодействии в CRM

### Resources and Links:
- Джоб сайты (LinkedIn, HeadHunter, и др.)
- CRM система для отслеживания коммуникаций
- Шаблоны ответов кандидатам
- Информация о вакансиях

### Instructions:

1. Установите регулярное время для проверки сообщений (например, утром, в обед и вечером)
2. После выходных в первую очередь обработайте все накопившиеся сообщения:
   - Отсортируйте по приоритету
   - Ответьте на срочные вопросы
   - Обработайте остальные сообщения по порядку
3. При ответе кандидатам:
   - Будьте вежливы и профессиональны
   - Предоставляйте полную информацию о вакансии
   - Отвечайте на все вопросы кандидата
   - Предлагайте следующие шаги (собеседование, дополнительная информация)
4. После каждого взаимодействия обновляйте информацию в CRM:
   - Отмечайте дату последнего контакта
   - Записывайте суть коммуникации
   - Обновляйте статус кандидата
5. Отслеживайте ответы кандидатов и продолжайте коммуникацию при необходимости

## Task 4: Применение полученных знаний по работе с договорами

### Steps:
1. Регулярно применять процедуры оформления договоров, изученные у Жени
2. Самостоятельно создавать договоры для новых сотрудников
3. Правильно заполнять все связанные таблицы
4. Проводить самопроверку перед финальным утверждением
5. При возникновении вопросов консультироваться с Женей
6. Документировать новые случаи и особенности для будущего использования

### Resources and Links:
- Процедуры оформления договоров от Жени
- Шаблоны договоров
- Таблицы для документооборота
- Контакты Жени для консультаций

### Instructions:

1. При создании каждого нового договора следуйте процедурам, изученным у Жени:
   - Используйте правильный шаблон для типа договора
   - Заполните все обязательные поля
   - Проверьте правильность ввода данных
2. При заполнении таблиц:
   - Введите информацию в правильные колонки
   - Проверьте формат данных (даты, суммы, контакты)
   - Убедитесь в соответствии данных между договором и таблицей
3. Перед финальным утверждением проведите самопроверку:
   - Проверьте все поля на заполненность
   - Убедитесь в отсутствии опечаток
   - Проверьте соответствие данных
4. Если возникают сомнения или вопросы, не стесняйтесь обращаться к Жене за консультацией
5. Ведите заметки о новых случаях или особенностях, которые могут быть полезны в будущем
6. Постепенно стремитесь к полной самостоятельности в работе с договорами
