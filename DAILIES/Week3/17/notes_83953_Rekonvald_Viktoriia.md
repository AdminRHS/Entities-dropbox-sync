### Step-by-Step Instructions from the Call

1. **Tool Setup**
   - Make sure you have **VS Code** and **Cursor** installed.
   - Log in with the correct accounts (HR / Dev / Admin, as allowed).
   - Enable the **Claude / Cloud extensions** in VS Code and Cursor.

2. **Avoid Heavy Tools at Start**
   - Do **not** run Windsurf for now (it loads the laptop like “150 Chrome tabs” and slows everything down).
   - Keep only the tools you really use open (VS Code, Cursor, browser).

3. **Video Research Workflow**
   - Take a chosen video (e.g., **Video_015** or other HR/automation videos from the Videos library).
   - Open the YouTube link or transcript in **Cursor** (or Claude) in a fresh chat/agent.
   - Scroll to the end of the transcript to see taxonomy / workflows, if available.

4. **Use Existing Prompts**
   - Go to the `PROMPTS` folders (Research / HR Automation / Video Processing) as Niko showed.
   - Copy the relevant **research or HR automation prompt**.
   - Paste it into **Gemini or Claude** and clearly explain:
     - What topic you want to research (e.g., “HR automation videos”, “Copilot + HR workflows”).
     - That you want step-by-step instructions or topic lists for new videos to process.

5. **Plan Mode in Cursor**
   - In Cursor, switch the side panel to **Plan Mode** (infinity symbol icon).
   - Tell Cursor in Plan Mode that you need: 
     - Topics for HR video instructions.
     - A list of video ideas / keywords to find on YouTube for HR automation.

6. **Connect Research Back to Task Manager**
   - For each promising video or topic:
     - Note the **tools, workflows, and tasks** it mentions (especially for HR).
     - Create or update entries in the **Task Manager** (tasks, subtasks, phases).
   - Mark which topics are **already covered** in the ecosystem and which are **new gaps**.

7. **Daily Working Rhythm**
   - Combine your regular HR work (candidates, calls, CRM) with:
     - **Short research blocks** (e.g., 30–60 minutes) for videos and prompts.
     - Documenting results directly into Task Manager or library files so you don’t lose context.

8. **Performance & Stability**
   - If the laptop starts freezing:
     - Close Windsurf and any extra heavy apps.
     - Keep a simple **tracking table in Excel/Sheets** (which prompts, which videos, which research session) as Niko suggested to avoid confusion when multiple processes run.
