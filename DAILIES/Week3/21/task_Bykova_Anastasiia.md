## Task 4: Create Social Media Carousels

### Steps:
1. Review content from completed courses
2. Identify key messages for social media posts
3. Design carousel slides in vertical format (9:16 or 4:5 ratio)
4. Create engaging first slide to grab attention
5. Develop 5-8 slides per carousel with clear message flow
6. Use consistent branding and design style
7. Add text overlays with key information
8. Export carousel as individual images
9. Import slides into Capcut or Canva
10. Add transitions and animations
11. Time transitions appropriately (2-3 seconds per slide)
12. Add background music if appropriate
13. Export final animated carousel
14. Review and test on mobile device
15. Submit for review or pass to video editors if needed

### Resources and Links:
- [Course Materials](https://drive.google.com/drive/folders/1r_JCEY1pv4y2Zv2sdgYQdYVnEI2tMS1P?usp=drive_link)
- Canva (free version)
- Capcut
- Adobe Firefly for image generation
- Company brand guidelines (if available)

### Instructions:

**Carousel Design Best Practices:**
- Start with an attention-grabbing first slide
- Use vertical format for Instagram/TikTok/Facebook Stories
- Keep text minimal and readable on mobile
- Use high-contrast colors for text visibility
- Include clear call-to-action on final slide
- Maintain consistent color scheme throughout
- Use templates for efficiency

**Animation Guidelines:**
- Keep transitions smooth and professional
- Don't make animations too fast or too slow
- Use consistent transition style throughout
- Add subtle motion to keep viewer engaged
- Test final video before submitting
- Export in highest quality available

**Self-Assessment Criteria:**
- Can I understand the message in 3 seconds per slide?
- Is text readable on a phone screen?
- Does the carousel tell a complete story?
- Are transitions smooth and professional?
- Does it align with brand style?
