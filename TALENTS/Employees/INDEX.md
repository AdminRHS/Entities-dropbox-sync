## Adding New Employees

### Step 1: Create Profile

1. Use template: [Employee_Profile_Schema.md](Employee_Profile_Schema.md)
2. Assign Employee ID from Finance Public
3. Name file: `EMP-{5-digit}.md` (e.g., `EMP-88105.md`)
4. Place in `Profiles/` folder

### Step 2: Add to Finance Public

1. Add employee to Finance Public master table
2. Include: ID, Name, Status, Rate, Department, Profession
3. Save and sync

### Step 3: Run Sync Automation

```bash
cd automations/employee-profile-manager
python3 sync_profiles.py
```

This will:
- Update profile with Finance Public data
- Regenerate INDEX.md
- Update taxonomy files

### Step 4: Create Work Folder (Optional)

For active employees:

```
Nov25/
└── {Department}/
    └── {Employee Name}/
        ├── Profile {Profession} {Name}.md
        ├── Week_1/
        ├── Week_2/
        └── ...
```

- **Daily Sync:** Automated at 6 AM UTC
