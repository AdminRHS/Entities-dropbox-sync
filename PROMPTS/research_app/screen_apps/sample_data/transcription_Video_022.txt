## Workflow: Build a Web Scraping Robot

**OBJECTIVE**: Extract a list of AI tools from a directory website without coding

**STEPS**:
1. Create account on Browse AI
2. Install Chrome extension
3. Click "Build New Robot"
4. Choose "Extract Structured Data" option
5. Enter target URL of website to scrape
6. Click "Start Recording"
7. Click robot icon and select "Capture List"
8. Click on first item to establish pattern
9. Click on second item - Browse AI auto-detects pattern
10. Map additional columns (Name, Description, Link)
11. Configure pagination (select "Next" button)
12. Click "Finish Recording"
13. Name the robot (e.g., "AI Tools Scraper")
14. Run task and specify number of pages to scrape
15. Wait for processing to complete
16. Export data as CSV or send to Google Sheets

**DURATION**: 5-10 minutes for setup, seconds for execution
**COMPLEXITY**: Low
**OUTPUT**: CSV file or Google Sheets with structured data
