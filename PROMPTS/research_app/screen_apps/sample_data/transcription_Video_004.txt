## Key Features

### Browser Capabilities
- **Built on Chromium**: Seamless Chrome extension compatibility
- **Multi-tab Context**: Can read and analyze content across multiple open tabs
- **Browser History Access**: Remembers past browsing for contextual assistance
- **One-click Summarization**: Instant webpage summarization button
- **Voice Mode**: Speak prompts instead of typing

### AI Assistant Features
- **Model Switching**: Force specific AI models with slash commands (/gpt, /claude, /gemini, /grok)
- **Custom Shortcuts**: Create reusable prompts with /command syntax
- **Advanced Sources**: Search web, academic papers, social discussions, finance, email, calendar
- **Agentic Control**: AI takes over browser to perform tasks autonomously

### Agentic Capabilities
- **Browser Automation**: Blue glow indicates AI is controlling browser
- **Multi-task Execution**: Can run multiple tasks across different tabs simultaneously
- **Form Filling**: Automatically fill out forms based on user input
- **Email Drafting**: Craft emails with context from past conversations
- **Calendar Management**: Find available times and create calendar events
- **Shopping Automation**: Compare products, read reviews, add items to cart

## Workflows Identified

### WORKFLOW 1: Summarize News from Multiple Sources
**OBJECTIVE**: Get a concise roundup of the most important news from newsletters and RSS feeds

**STEPS**:
1. Open news aggregation tool (e.g., Feedly)
2. Activate Comet Assistant
3. Provide prompt to skim all news and create roundup
4. Agent clicks into each newsletter/article
5. Agent compiles summary of news mentioned across multiple sources
6. Present final categorized roundup

**COMPLEXITY**: Medium
**TOOLS USED**: Perplexity Comet, Feedly, RSS feeds

### WORKFLOW 2: Repurpose YouTube Video into Blog Post
**OBJECTIVE**: Transform video content into written blog article

**STEPS**:
1. Navigate to YouTube video
2. Open Assistant and request 500-word blog post about video content
3. AI analyzes video transcript and key points
4. Generates formatted blog post with key takeaways
5. Copy content directly to blog

**COMPLEXITY**: Low
**TOOLS USED**: Perplexity Comet, YouTube

### WORKFLOW 3: Automated Calendar Scheduling
**OBJECTIVE**: Find optimal meeting time and create calendar event

**STEPS**:
1. Open Google Calendar
2. Provide meeting parameters (person, duration, time preferences, day restrictions)
3. AI reviews calendar for conflicts
4. AI identifies best available time slot
5. AI creates calendar event with correct details

**COMPLEXITY**: Medium
**TOOLS USED**: Perplexity Comet, Google Calendar

### WORKFLOW 4: Document Proofreading with Inline Comments
**OBJECTIVE**: Proofread Google Doc and add suggested edits as comments

**STEPS**:
1. Open Google Doc
2. Request proofreading with inline comments
3. AI takes browser control
4. AI highlights paragraphs and adds margin comments with suggestions
5. Review all suggested changes
6. Optionally: Switch to suggesting mode and apply changes directly

**COMPLEXITY**: High
**TOOLS USED**: Perplexity Comet, Google Docs

## Notes

Comprehensive AI browser demonstration showing practical business and productivity use cases. Comet can replace multiple AI subscriptions (ChatGPT, Claude, Gemini, Grok) with single $20/month plan. Notable features: multi-tab context, custom shortcuts, agentic browser control, parallel task execution. Limitations: Speed (slower than manual for simple tasks), some features still buggy (email drafting, restaurant booking). Best use case: Running multiple tasks in background while you work on other things. Built on Chromium for full Chrome extension compatibility.
