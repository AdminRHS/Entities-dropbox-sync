# Brochure Design Generation

**Purpose**: Generate brochure designs for clients by analyzing website and existing materials to create new design proposals.

**Version**: 1.0  
**Date**: 2025-11-21  
**Status**: Active  
**Related Prompts**: None

---

## Overview

This prompt enables designers to create new brochure/leaflet designs by analyzing client websites and existing materials. It uses Perplexity Labs for comprehensive analysis and design generation.

**Primary Use Case**: Creating brochure design proposals for clients  
**Target Platform**: Perplexity Labs  
**Department**: DGN (Design)

---

## Prompt Instructions

### Base Prompt

```
Hello! I need to design a proposal of a new leaflet or brochure design for a client. Here's their website [WEBSITE_URL]
They also have a current leaflet [LEAFLET_URL]
Please analyze the information and the design style of the website and of the old leaflet, and generate a new version of a leaflet or a brochure for them
```

### Usage Instructions

1. **Gather Materials**:
   - Client website URL
   - Current leaflet/brochure URL (if available)

2. **Replace Placeholders**:
   - `[WEBSITE_URL]` with actual website URL
   - `[LEAFLET_URL]` with current leaflet URL

3. **Execute in Perplexity Labs**: Run the prompt with materials attached

4. **Review Output**: Verify design aligns with brand style and requirements

### Required Materials
