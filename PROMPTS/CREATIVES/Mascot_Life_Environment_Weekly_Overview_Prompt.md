# Mascot Life Environment Association with Company Weekly Overview - Prompt Template

**Purpose:** Create mascot visual environments that reflect company weekly activities, strategic decisions, and cultural shifts through natural habitat associations  
**Version:** 1.0  
**Created:** December 25, 2025  
**Department:** Design / Creative  
**Use Case:** Game Academy course covers, company announcements, weekly reports visualization

---

## Overview

This prompt template enables designers to create mascot visual environments that metaphorically represent Remote Helpers' weekly activities, strategic initiatives, and organizational culture. By associating company themes with mascot natural environments, we create visual narratives that reinforce company values and achievements.

---

## Base Prompt Template

### Standard Format

```
A creative 3D cartoon Pixar-like cover image featuring [Mascot Name] in their natural environment. 
In this scene they are [Mascot Activity] at/in/around [Location], with [Clothing Accent] added to 
their usual outfit. NO TEXT on the image. Cover image is square.
```

### Enhanced Format with Weekly Overview Association

```
A creative 3D cartoon Pixar-like cover image featuring [Mascot Name] in their natural environment. 
In this scene they are [Mascot Activity] at/in/around [Location], with [Clothing Accent] added to 
their usual outfit. The environment reflects [Weekly Theme/Strategic Initiative], showing 
[Metaphorical Elements] that represent [Company Activity/Decision]. NO TEXT on the image. 
Cover image is square.
```

## Prompt Construction Workflow

### Step 1: Identify Weekly Theme
1. Review weekly overview document (W2+W3_Niko files)
2. Identify primary strategic theme or cultural shift
3. Select relevant theme category from mapping above
4. Note specific company activities or decisions

### Step 2: Select Mascot
1. Choose appropriate mascot for department/project:
   - **Falcon:** UI/UX Design, Game Academy courses
   - **Shark:** General company, common projects
   - **Hummingbird:** Design department, creative projects
   - **Chameleon:** Video department, adaptation themes
2. Consider mascot's natural behaviors that align with theme

### Step 3: Map Theme to Natural Environment
1. Identify mascot's natural habitat
2. Find natural behaviors that metaphorically represent company activity
3. Select location that reinforces theme
4. Choose activities that align with company work

### Step 4: Add Metaphorical Elements
1. Identify visual elements that represent company concepts
2. Integrate elements naturally into mascot's environment
3. Ensure elements are recognizable but not literal
4. Maintain natural habitat authenticity

### Step 5: Create Clothing Accent
1. Design accessory that reinforces theme
2. Keep it subtle and integrated
3. Ensure it doesn't distract from natural environment
4. Make it relevant to company activity

### Step 6: Construct Final Prompt
1. Use enhanced format template
2. Fill in all variables:
   - [Mascot Name]
   - [Mascot Activity]
   - [Location]
   - [Clothing Accent]
   - [Weekly Theme/Strategic Initiative]
   - [Metaphorical Elements]
   - [Company Activity/Decision]
3. Ensure NO TEXT instruction is included
4. Specify square format

## Quality Guidelines

### Natural Environment Authenticity
- ✅ Mascot must be in their actual natural habitat
- ✅ Environment must be believable and natural
- ✅ Activities must align with mascot's real behaviors
- ❌ Avoid artificial or unrealistic settings
- ❌ Don't force company elements unnaturally

### Metaphorical Integration
- ✅ Company themes represented through natural elements
- ✅ Visual metaphors should be subtle but recognizable
- ✅ Elements should enhance, not distract from natural scene
- ❌ Avoid literal representations (no computers, no office elements)
- ❌ Don't break natural habitat authenticity

### Clothing Accents
- ✅ Small, integrated accessories
- ✅ Relevant to theme but not overwhelming
- ✅ Natural colors that complement mascot
- ❌ Avoid large or distracting accessories
- ❌ Don't cover mascot's natural features

### Composition
- ✅ Square format (1:1 aspect ratio)
- ✅ Mascot as focal point
- ✅ Environment supports narrative
- ✅ Balanced composition
- ❌ No text overlays
- ❌ No company logos (unless specifically requested)

## Usage Instructions

### For Designers Creating Course Covers

1. **Review Weekly Overview:**
   - Read relevant W2+W3_Niko overview file
   - Identify primary theme for the week
   - Note key strategic decisions or cultural shifts

2. **Select Theme Category:**
   - Match weekly theme to category mapping above
   - Review example prompts for that category
   - Note metaphorical elements used

3. **Choose Mascot:**
   - Select appropriate mascot (Falcon for UI/UX courses)
   - Consider mascot's natural behaviors
   - Ensure alignment with theme

4. **Build Prompt:**
   - Use enhanced format template
   - Fill in all variables
   - Ensure natural environment authenticity
   - Verify metaphorical integration

5. **Generate Image:**
   - Use ChatGPT (Developers account recommended)
   - Generate multiple variations
   - Select best composition
   - Refine if needed

6. **Document:**
   - Record prompt in mascot table
   - Note unique location and clothing accent
   - Track for uniqueness (avoid duplicates)

### For Company Announcements

1. **Identify Announcement Theme:**
   - Determine primary message
   - Match to weekly theme if relevant
   - Select appropriate mascot

2. **Create Prompt:**
   - Use standard or enhanced format
   - Include announcement elements metaphorically
   - Maintain natural environment

3. **Generate and Refine:**
   - Create initial variations
   - Select best representation
   - Ensure brand consistency
