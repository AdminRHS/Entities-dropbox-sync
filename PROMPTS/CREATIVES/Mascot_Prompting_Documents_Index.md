#### Redesigning Cover Images for Game Academy
- **Prompt Template Format:**
  ```
  A creative 3D cartoon Pixar-like cover image featuring [Mascot Name] in their natural environment. 
  In this scene they are [Mascot Activity] at/in/around [Location], with [Clothing Accent] added 
  to their usual outfit. NO TEXT on the image. Cover image is square.
  ```

- **Example:**
  ```
  A creative 3D cartoon Pixar-like cover image featuring Sharky the Shark in his natural environment. 
  In this scene he is swimming past seashells projecting glowing holograms inside a kelp cavern, 
  wearing his regular beige shirt and cargo shorts plus a bright red scarf. Cover image is square.
  ```

- **Key Requirements:**
  - Square format
  - NO TEXT on image
  - Natural environment setting
  - Unique clothing accent for each image
  - Mascot and location remain consistent
  - Add software/tool icons if course is about specific tools

- **Workflow Notes:**
  - FIRST provide clothing accent and full prompt
  - THEN generate picture immediately
  - Come up with new clothing accent for every new image
  - Use mascots provided, not just generic animals
  - Incorporate text and icons neatly and creatively

## Mascot Prompting Guidelines Summary

### Core Template Structure

1. **Format:**
   - 3D cartoon Pixar-like style
   - Natural environment setting
   - Mascot activity description
   - Location specification
   - Clothing accent addition
   - NO TEXT instruction
   - Square format specification

2. **Key Elements:**
   - Mascot name (Falcon, Shark, Hummingbird, Chameleon, etc.)
   - Natural environment (habitat-specific)
   - Activity (behavior-specific)
   - Location (environment-specific)
   - Clothing accent (unique per image)
   - Optional: Tool/software icons for tool-specific courses

3. **Quality Requirements:**
   - Natural environment authenticity
   - Consistent mascot appearance
   - Unique clothing accents
   - Unique locations
   - No text overlays
   - Square format (1:1)

## Usage Workflow

### For Course Cover Creation

1. **Select Course Theme**
   - Review course content
   - Identify primary learning objective
   - Match to weekly theme if applicable

2. **Choose Mascot**
   - Select appropriate mascot (Falcon for UI/UX)
   - Review mascot's natural behaviors
   - Consider theme alignment

3. **Generate Unique Elements**
   - Create unique location (check tracking table)
   - Design unique clothing accent (check tracking table)
   - Ensure no duplicates

4. **Build Prompt**
   - Use template format
   - Fill in all variables
   - Add tool icon if course-specific

5. **Generate Image**
   - Use ChatGPT (Developers account)
   - Generate multiple variations
   - Select best composition

6. **Document**
   - Add to mascot tracking table
   - Record location and clothing accent
   - Save prompt for reference

## Key Learnings from Design Documents

1. **Iteration is Essential:**
   - Multiple generation stages
   - Clothing updates
   - Composition refinements
   - Detail replacements

2. **Uniqueness Tracking:**
   - Document all locations used
   - Track all clothing accents
   - Prevent duplicates
   - Maintain variety

3. **Natural Environment Priority:**
   - Authentic habitats
   - Realistic behaviors
   - Natural compositions
   - Avoid artificial elements

4. **Account Management:**
   - Use separate accounts for different tasks
   - Developers account for image generation
   - Personal account for descriptions
   - Prevent conversation fatigue
