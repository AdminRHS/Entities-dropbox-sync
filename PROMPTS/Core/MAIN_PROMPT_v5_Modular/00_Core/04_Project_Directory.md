## Important Usage Note

When processing meeting transcripts, use this directory to **accurately identify mentioned projects**. Match project names, abbreviations, keywords, or contextual clues to specific active projects.

**Matching Strategies:**
- Project abbreviations (RH, MDL, DGN, l-gn, v-mg, etc.)
- Full project names
- Platform mentions (Dribbble, Behance, LinkedIn, etc.)
- Client names or industries
- Tool mentions (Figma files, Google Drive folders)
- Resource URLs

**Confidence Levels:**
- **High:** Exact project name/abbreviation + context confirms
- **Medium:** Keyword match or context-based inference
- **Low:** Unclear match, requires manual verification

## Recognition Patterns

### High-Confidence Matches
- Exact project abbreviation mentioned (e.g., "MDL-EU project")
- Project name explicitly stated (e.g., "Models EU campaign")
- Unique combination of platform + department (e.g., "Dribbble portfolio" = DGN-DRB)

### Medium-Confidence Matches
- Platform mention + general context (e.g., "LinkedIn outreach" could be LI-SMM or l-gn)
- Tool + department context (e.g., "Figma" + Design team = likely DGN or UX-DGN)
- Client industry clues (e.g., "modeling agencies" = MDL-EU or MDL-UA)

### Low-Confidence Matches (Requires Clarification)
- Generic terms without context (e.g., "the project")
- Multiple possible projects (e.g., "social media post" could be multiple SMM projects)
- New/unlisted projects (flag for manual review)
