## 3. Processes Library (Sales Workflows)

### PROC-SALES-001: Sales Pipeline Management

**Steps:**
1. **Lead Generation** - Inbound/outbound leads
2. **Qualification** - BANT (Budget, Authority, Need, Timeline)
3. **Discovery Call** - Understand client needs
4. **Proposal** - Prepare custom proposal
5. **Negotiation** - Discuss terms, pricing
6. **Close** - Sign contract, collect payment
7. **Handoff** - Transfer to delivery team

**Metrics:**
- Conversion rate: Lead → Client
- Average deal size
- Sales cycle length

### PROC-SALES-002: Client Onboarding

**Steps:**
1. **Kickoff Call** - Introduce team, set expectations
2. **Requirements Gathering** - Detailed brief
3. **Timeline Agreement** - Set milestones
4. **Access Setup** - Tools, accounts, platforms
5. **First Deliverable** - Set tone for project

**Goal:** Smooth start, client confidence

### PROC-SALES-003: Proposal Creation

**Steps:**
1. **Understand Scope** - Client requirements
2. **Select Services/Products** - Match to needs
3. **Pricing** - Calculate based on scope
4. **Write Proposal** - Professional document
5. **Present** - Walk through with client
6. **Follow Up** - Answer questions, close

**Template Sections:**
- Executive Summary
- Scope of Work
- Deliverables
- Timeline
- Pricing
- Terms & Conditions
