### PROC-AI-001: Prompt Development Cycle

**Process:** Creating and refining AI prompts
**Duration:** 1-5 days (depending on complexity)
**Owner:** Prompt Engineers (Matvii, Vladislav)

**Steps:**
1. Define objective (what the prompt should achieve)
2. Draft initial prompt
3. Test with Claude/ChatGPT
