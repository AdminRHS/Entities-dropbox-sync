# Subscription Announcement Creation

**Purpose**: Create announcement posts with formatted text and mascot-based images for multiple departments (Global News, Design, Video).

**Version**: 1.0  
**Date**: 2025-11-21  
**Status**: Active

## Overview

This prompt enables creation of subscription announcement posts with formatted text and custom images featuring department mascots. The workflow includes multiple iterations to refine image generation and ensure proper mascot representation.

**Primary Use Case**: Creating announcement posts for new subscription additions  
**Target Platform**: ChatGPT (text) + Midjourney/DALL-E (images)  
**Department**: Multi (Global News, Design, Video departments)

---

## Prompt Instructions

### Step 1: Initial Request

```
Hey chat! i'm a designer and got an assignment to create an announcement post. [ANNOUNCEMENT_TEXT_IN_RUSSIAN/ENGLISH]

I need the same nicely formatted text of the announcement for all three chats, and three pictures. On each pictures there has to be the mascot (shark for common, hummingbird for design, and chameleon for videographers) in their natural habitat, the icon of Claude AI, and text "Subscriptions Added", all nicely incorporated into what's going on on the picture. What's going on on the picture has to correspond the announcement of new subscriptions. Aspect ratio of images is 16:9. I'll give you materials for mascots, and then you generate
```

### Step 2: Mascot Clarification

```
hey, one image has to have the shark, another the hummingbird, and the third the chameleon. and here's the claude logo
```

### Step 3: Creative Refinement

```
don't use drawing boards, screens, or gadgets on pictures. come up with something more creative, appropriate and incorporated into the mascot's habitat
```

### Step 4: Final Refinement

```
adding to my previous edits, use mascots that i gave you, not just animals. and incorporate text and claude icon neatly and creatively into the composition. please generate all three pictures again
```

## Usage Workflow

### Step 1: Prepare Announcement
1. Write announcement text (can be in Russian/Ukrainian/English)
2. Identify key information:
   - Subscription details
   - Account information
   - Plan types

### Step 2: Generate Text
1. Request formatted text for all three chats
2. Review and refine formatting
3. Ensure consistency across versions

### Step 3: Generate Images
1. Provide mascot references
2. Provide Claude logo
3. Execute image generation with requirements
4. Iterate based on feedback

### Step 4: Final Review
1. Verify text formatting
2. Check image quality
3. Ensure mascot accuracy
4. Verify creative integration

## Image Requirements

### Format Specifications
- **Aspect Ratio**: 16:9
- **Mascots Required**:
  - Shark (Global News/common)
  - Hummingbird (Design)
  - Chameleon (Video/Videographers)
- **Elements Required**:
  - Claude AI icon
  - Text: "Subscriptions Added"
  - Mascot in natural habitat

### Creative Guidelines
- ✅ Incorporate elements naturally into habitat
- ✅ Avoid drawing boards, screens, gadgets
- ✅ Use provided mascot references (not generic animals)
- ✅ Creative composition with text and icon
- ❌ Don't use generic animal images
- ❌ Don't include technology/gadgets
- ❌ Don't place elements awkwardly

- ✅ Avoid technology/gadgets in images

- ❌ Don't place elements separately from scene
