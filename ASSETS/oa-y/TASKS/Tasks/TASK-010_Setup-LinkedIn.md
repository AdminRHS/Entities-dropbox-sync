# TASK: Setup LinkedIn Account

ID: TASK-010

Category: Onboarding/Professional/LinkedIn

Tags: [onboarding, linkedin, networking, lead-generation, professional-profile]

## Objective

Create and optimize a professional LinkedIn account for lead generation, networking, and professional presence. LinkedIn is essential for Lead Generation team members and used by other departments for professional networking.

## Steps

1. **Create LinkedIn Account**
   - Visit https://www.linkedin.com/signup
   - Enter work Gmail address (from TASK-009)
   - Create strong password
   - Enter first and last name
   - Complete initial registration
   - Verify email address

2. **Complete Basic Profile (Day 1)**
   - Add professional headline
   - Select location
   - Choose industry
   - Add current position (if applicable)
   - Complete basic profile information

3. **Upload Professional Photo**
   - Upload headshot (400x400+ pixels minimum)
   - Photo should be:
     - Professional headshot
     - Clear face visible
     - Professional attire
     - Neutral background
   - Crop and adjust as needed

4. **Add Background Banner**
   - Upload professional banner image (1584×396 pixels)
   - Use company-provided banner if available
   - Or create professional banner
   - Ensure image is high quality

5. **Write Professional About Section**
   - Write 2-3 paragraphs about yourself
   - Include:
     - Professional background
     - Skills and expertise
     - What you're looking for
   - Use ChatGPT for assistance if needed
   - Keep it professional and engaging

6. **Add Work Experience**
   - Add current position at Remote Helpers
   - Backdate start date 2-3 months if new
   - Add job title: "Account Manager" or appropriate title
   - Write professional description
   - Add previous relevant experience

7. **Add Education**
   - Add all educational institutions
   - Include degrees and certifications
   - Add relevant courses
   - Complete education section fully

8. **Add Skills**
   - Add relevant professional skills
   - Pin key skills:
     - Lead Generation
     - Account Management
     - Research Skills
   - Add other relevant skills
   - Get skill endorsements from connections

9. **Configure Privacy Settings**
   - Review privacy settings
   - Configure profile visibility
   - Set connection preferences
   - Configure activity visibility
   - Review data settings

10. **Build Connections Gradually**
    - Week 1: Add 15-20 connections per day
    - After Week 1: Add 20-25 connections per day
    - Connect with colleagues
    - Connect with industry professionals
    - Avoid mass connection requests

11. **Set Up LinkedIn Extensions**
    - Install RE Extension in Chrome
    - Install SalesQL if applicable
    - Configure extensions with LinkedIn
    - Test extension functionality

12. **Complete Profile Optimization**
    - Add all relevant sections
    - Complete profile to 100%
    - Add portfolio links if applicable
    - Add certifications
    - Keep profile updated

## Common Issues and Solutions

**Issue: Account restricted or flagged**
- Avoid using VPN during setup
- Build connections gradually
- Use stable IP address
- Complete ID verification if requested
- Contact LinkedIn support if needed

**Issue: Cannot upload photo**
- Verify photo meets size requirements (400x400+)
- Check file format (JPG, PNG)
- Ensure file size is reasonable
- Try different browser if issues persist

**Issue: Extensions not working**
- Verify extensions are installed in Chrome
- Check extension permissions
- Ensure LinkedIn page is loaded
- Restart browser if needed
