# TASK: Setup Claude AI (Desktop + Code Extension)

ID: TASK-014

Category: Onboarding/AI/Claude

Tags: [onboarding, ai, claude, claude-desktop, claude-code, development]

## Objective

Install and configure Claude Desktop application and Claude Code extension for VS Code/Cursor. Claude AI provides advanced reasoning, extended thinking, and development workflows essential for professional AI-assisted work.

## Steps

### Part 1: Claude Desktop Setup

1. **Create Anthropic Account**
   - Visit https://claude.ai/
   - Click "Sign up" or "Get started"
   - Enter email address (work email recommended)
   - Create password
   - Verify email address
   - Complete account setup

2. **Choose Claude Plan**
   - Review available plans:
     - Free tier (limited)
     - Pro tier (recommended for work)
   - Select appropriate plan
   - Complete payment if upgrading
   - Verify subscription status

3. **Download Claude Desktop**
   - Visit Claude Desktop download page
   - Select your operating system
   - Download Claude Desktop installer
   - Run the installer
   - Complete installation

4. **Sign In to Claude Desktop**
   - Open Claude Desktop application
   - Sign in with Anthropic account
   - Verify authentication
   - Complete initial setup

5. **Configure Claude Desktop Settings**
   - Access Settings
   - Configure preferences:
     - Model selection (Claude 4 Sonnet recommended)
     - Response settings
     - Privacy settings
   - Set up custom instructions
   - Configure data controls

6. **Learn Claude Desktop Features**
   - Understand chat interface
   - Learn extended thinking mode
   - Practice file uploads
   - Understand conversation management
   - Learn keyboard shortcuts

### Part 2: Claude Code Extension Setup

7. **Install Claude Code Extension**
   - Open VS Code or Cursor
   - Go to Extensions (Ctrl+Shift+X)
   - Search for "Claude Code" or "Claude"
   - Find "Claude Code" by Anthropic
   - Click "Install"
   - Wait for installation

8. **Authenticate Claude Code Extension**
   - After installation, look for Claude icon
   - Click Claude icon in sidebar or top-right
   - Sign in with Anthropic account
   - Complete authentication flow
   - Verify subscription if required

9. **Configure Claude Code Settings**
   - Open VS Code/Cursor Settings
   - Search for "Claude" settings
   - Configure extension preferences:
     - Default model
     - Response behavior
     - Context settings
   - Set up workspace-specific settings

10. **Test Claude Code Functionality**
    - Open any file in workspace
    - Click Claude icon
    - Type test prompt
    - Verify response appears
    - Test code assistance features

11. **Set Up MCP Integration (Advanced)**
    - Learn about Model Context Protocol
    - Set up MCP servers if needed
    - Configure MCP connections
    - Test MCP functionality

## Common Issues and Solutions

**Issue: Cannot create Anthropic account**
- Verify email is valid
- Check if account already exists
- Try different email if needed
- Contact Anthropic support if issues persist

**Issue: Claude Code extension not working**
- Verify VS Code/Cursor is installed
- Check extension is installed correctly
- Restart IDE after installation
- Verify authentication completed
- Check subscription status

**Issue: Cannot authenticate extension**
- Verify Anthropic account is active
- Check subscription status
- Try signing out and back in
- Reinstall extension if needed
