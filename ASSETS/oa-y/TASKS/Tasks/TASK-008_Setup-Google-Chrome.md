# TASK: Setup Google Chrome

ID: TASK-008

Category: Onboarding/Browser/Chrome

Tags: [onboarding, browser, chrome, productivity, extensions]

## Objective

Install and configure Google Chrome with a work profile, sync settings, and essential extensions. Chrome is the primary browser used for all work activities including LinkedIn, CRM access, and web-based tools.

## Steps

1. **Download Google Chrome**
   - Visit https://www.google.com/chrome/
   - Click "Download Chrome"
   - Select your operating system
   - Download the installer

2. **Install Chrome**
   - Run the downloaded installer
   - Follow installation wizard
   - Accept terms and conditions
   - Complete installation
   - Chrome will launch automatically

3. **Create Work Chrome Profile**
   - Click profile icon (top-right corner)
   - Click "Add" or "Manage people"
   - Click "Add person"
   - Name profile: `LinkedIn_<FirstName>` or `Work_<FirstName>`
   - Choose profile icon/color
   - Create profile

4. **Sign In to Chrome Profile**
   - In your work profile, click "Turn on sync"
   - Sign in with work Gmail account
   - Enable sync for:
     - ✅ Bookmarks
     - ✅ History
     - ✅ Passwords
     - ✅ Extensions
     - ✅ Settings
   - Complete sync setup

5. **Configure Chrome Settings**
   - Go to Settings (chrome://settings/)
   - Set default search engine: Google
   - Configure homepage (company dashboard if applicable)
   - Set download location
   - Configure privacy settings

6. **Set Up Bookmarks**
   - Bookmark essential work sites:
     - CRM system
     - Discord web
     - Dropbox web
     - LinkedIn
     - Google Workspace (Drive, Sheets, Gmail)
     - Company resources
   - Organize bookmarks in folders

7. **Install Essential Extensions**
   - Open Chrome Web Store
   - Install required extensions:
     - **Google Scrape** (for Lead Generation)
     - **RE Extension** (LinkedIn extension)
     - **SalesQL** (if applicable)
     - **Password Manager** (if using)
   - Pin extensions to toolbar

8. **Configure Tab Management**
   - Learn tab grouping
   - Set up tab groups for different work areas
   - Pin frequently used tabs
   - Configure tab search functionality

9. **Set Up Chrome Sync**
   - Verify sync is working
   - Check sync status in Settings
   - Test accessing bookmarks from another device
   - Ensure extensions sync across devices

10. **Configure Security Settings**
    - Enable Safe Browsing
    - Configure site permissions
    - Set up password saving preferences
    - Configure autofill settings

11. **Test Browser Functionality**
    - Test opening CRM
    - Test accessing LinkedIn
    - Test extension functionality
    - Verify bookmarks work
    - Test sync across devices
