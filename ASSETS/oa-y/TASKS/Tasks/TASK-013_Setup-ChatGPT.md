# TASK: Setup ChatGPT

ID: TASK-013

Category: Onboarding/AI/ChatGPT

Tags: [onboarding, ai, chatgpt, gpt-5, ai-assistant]

## Objective

Create ChatGPT account and learn to use GPT-5 and advanced features for content generation, research, coding assistance, and professional workflows. ChatGPT is one of the primary AI tools used across all departments.

## Steps

1. **Create ChatGPT Account**
   - Visit https://chat.openai.com/
   - Click "Sign up"
   - Enter email address (work email recommended)
   - Create password
   - Verify email address
   - Complete account setup

2. **Choose ChatGPT Plan**
   - Review available plans:
     - Free tier (limited)
     - Plus/Pro tier (recommended for work)
   - Select appropriate plan
   - Complete payment if upgrading
   - Verify subscription status

3. **Access GPT-5 and Advanced Models**
   - Navigate to model selector
   - Select GPT-5 (if available)
   - Understand model differences:
     - GPT-4o
     - GPT-4 Turbo
     - GPT-5 (latest)
   - Learn when to use each model

4. **Configure ChatGPT Settings**
   - Go to Settings
   - Configure preferences:
     - Model selection
     - Temperature settings
     - Response length
   - Set up custom instructions
   - Configure data controls

5. **Learn Basic Features**
   - Understand chat interface
   - Learn to write effective prompts
   - Practice conversation flow
   - Learn to edit and regenerate responses
   - Understand conversation history

6. **Explore Tools and Connectors**
   - Access Tools & Connectors section
   - Set up web browsing
   - Configure code interpreter
   - Set up file uploads
   - Configure integrations

7. **Set Up Agent Capabilities**
   - Learn about ChatGPT Agents
   - Understand agent workflows
   - Set up automated tasks
   - Configure agent instructions
   - Test agent functionality

8. **Learn Voice and Video Features**
   - Set up voice conversations
   - Configure video capabilities
   - Test multimedia features
   - Understand voice settings

9. **Create Custom GPTs**
   - Access GPT Builder
   - Create custom GPT for specific tasks
   - Configure GPT instructions
   - Add knowledge base
   - Test custom GPT

10. **Set Up Professional Workflows**
    - Learn advanced prompting techniques
    - Set up templates for common tasks
    - Configure workflow automation
    - Integrate with other tools
    - Optimize for your use cases

11. **Understand Usage Limits**
    - Check message limits
    - Understand rate limits
    - Monitor usage dashboard
    - Plan usage accordingly

## Requirements

- Accounts needed:

  - Email address (work email recommended)
  - Payment method (for Plus/Pro plans)
- Tools needed:
  - Internet connection
  - Browser (Chrome recommended)
- Prerequisites:
  - None (can be set up independently)

## Common Issues and Solutions

**Issue: Cannot create account**
- Verify email is valid
- Check if account already exists
- Try different email if needed
- Contact OpenAI support if issues persist

**Issue: Cannot access GPT-5**
- Verify subscription tier
- Check if GPT-5 is available in your region
- Update ChatGPT app if using mobile
- Contact support if access issues

**Issue: Rate limits reached**
- Wait for limit reset
- Upgrade plan if needed
- Optimize prompt usage
- Use alternative AI tools temporarily
