# TASK: Setup Perplexity AI

ID: TASK-015

Category: Onboarding/AI/Perplexity

Tags: [onboarding, ai, perplexity, research, web-search]

## Objective

Create Perplexity AI account and learn to use it for quick research, fact-checking, API documentation lookups, and real-time information gathering. Perplexity is excellent for research tasks and complements other AI tools.

## Steps

1. **Create Perplexity Account**
   - Visit https://www.perplexity.ai/
   - Click "Sign up" or "Get started"
   - Choose sign-up method:
     - Email sign-up
     - Google account (work Gmail)
     - Apple ID
   - Complete account creation
   - Verify email if required

2. **Choose Perplexity Plan**
   - Review available plans:
     - Free tier (limited searches)
     - Pro tier (unlimited, advanced features)
   - Select appropriate plan
   - Complete payment if upgrading
   - Verify subscription status

3. **Familiarize Yourself with Interface**
   - Understand search interface
   - Learn to ask questions
   - Understand citation system
   - Learn to view sources
   - Practice basic searches

4. **Configure Settings**
   - Access Settings/Preferences
   - Configure search preferences
   - Set default model if available
   - Configure citation preferences
   - Set up notification settings

5. **Learn Advanced Features**
   - Understand Pro Search mode
   - Learn to use focus modes:
     - Academic
     - Writing
     - Research
     - Code
   - Practice advanced queries
   - Learn citation navigation

6. **Set Up Collections**
   - Create research collections
   - Save important searches
   - Organize research topics
   - Share collections if needed
   - Manage saved content

7. **Learn Research Workflows**
   - Practice fact-checking
   - Research companies/people
   - Look up API documentation
   - Find technical information
   - Verify information sources

8. **Integrate with Workflow**
   - Use Perplexity for quick lookups
   - Complement other AI tools
   - Use for API/field documentation
   - Research before using ChatGPT/Claude
   - Fact-check AI responses

9. **Set Up Mobile App (Optional)**
   - Download Perplexity mobile app
   - Sign in with account
   - Configure mobile settings
   - Test mobile functionality
   - Set up notifications

10. **Learn Best Practices**
    - Write effective queries
    - Use specific questions
    - Check citations
    - Verify information
    - Combine with other tools

## Requirements

- Accounts needed:
  - Perplexity account (created during setup)
  - Email or Google account
  - Payment method (for Pro plans)
- Tools needed:

  - Browser (Chrome recommended)

  - None (can be set up independently)

## Common Issues and Solutions

**Issue: Cannot create account**
- Verify email is valid
- Try different sign-up method
- Check if account already exists
- Contact Perplexity support if needed

**Issue: Search limits reached**
- Wait for limit reset (free tier)
- Upgrade to Pro for unlimited
- Optimize search queries
- Use other AI tools temporarily

**Issue: Citations not showing**
- Verify you're using Pro Search
- Check if sources are available
- Try rephrasing query
- Check subscription status
