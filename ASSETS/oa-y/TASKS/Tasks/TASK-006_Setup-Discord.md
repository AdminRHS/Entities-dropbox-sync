# TASK: Setup Discord

ID: TASK-006

Category: Onboarding/Communication/Discord

Tags: [onboarding, communication, discord, team-collaboration, voice-channels]

## Objective

Install and configure Discord as the primary team communication platform. Discord is used daily for team communication, receiving tasks, joining voice channels, staying online during work hours, and receiving automated CRM notifications.

## Steps

1. **Download Discord**
   - Visit https://discord.com/download
   - Select your operating system (Windows/Mac/Linux)
   - Download the Discord installer
   - Run the installer and follow setup wizard

2. **Create Discord Account**
   - Open Discord application
   - Click "Register" if you don't have an account
   - Enter your email address (use work email if available)
   - Create a username (use your real name or professional handle)
   - Create a strong password
   - Complete email verification if required

3. **Join Company Workspace Server**
   - Receive Discord server invite link from HR
   - Click the invite link or paste it in Discord
   - Accept server invitation
   - Verify you can see the company server in your server list

4. **Complete Server Registration**
   - Navigate to registration channel (usually `#onboarding` or `#welcome`)
   - Follow instructions to register your account
   - May need to verify your identity or complete onboarding steps
   - Wait for role assignment from HR/admin

5. **Upload Profile Photo**
   - Click on your profile icon (bottom-left)
   - Select "Edit User Profile"
   - Upload a professional photo (headshot recommended)
   - Photo should be clear and professional
   - Save changes

6. **Configure Notification Settings**
   - Click server name → Notification Settings
   - Configure notification preferences:
     - **@mentions:** Always notify
     - **All Messages:** Configure based on your preference
     - **Nothing:** Only for channels you want to mute
   - Set up notification sounds if desired
   - Configure mobile notifications if using Discord mobile

7. **Join Required Channels**
   - Navigate to main channels:
     - `#general` - General announcements
     - `#onboarding` - Onboarding information
     - `#onboarding-selfie` - Upload your selfie here
     - Department-specific channels (HR, Dev, Sales, etc.)
   - Read channel descriptions to understand purpose
   - Join voice channels when needed for calls

8. **Set Up Status and Availability**
   - Set your status to "Online" during work hours
   - Configure custom status if desired
   - Set "Do Not Disturb" during breaks if needed
   - Ensure status reflects your availability

9. **Configure Voice & Video Settings**
   - Go to User Settings → Voice & Video
   - Test your microphone and speakers
   - Configure input/output devices
   - Set up push-to-talk if preferred
   - Test video settings if using video calls

10. **Learn Key Features**
    - **Text Channels:** For written communication
    - **Voice Channels:** For voice calls and meetings
    - **Direct Messages:** For private conversations
    - **@mentions:** To notify specific users
    - **Threads:** For organized discussions
    - **File Sharing:** Upload files directly to channels

11. **Upload Onboarding Selfie**
    - Navigate to `#onboarding-selfie` channel
    - Upload a clear selfie (face visible)
    - Background not critical
    - This is used to create your official styled photo

12. **Verify CRM Integration**
    - Check if Discord ID is added to CRM profile
    - Verify automated notifications are working
    - Test receiving CRM reminders via Discord
    - Contact HR if notifications aren't working

- Tools needed:

  - Webcam (optional, for video calls)

  - Understanding of basic communication tools

## Common Issues and Solutions

**Issue: Cannot join server**
- Verify invite link is valid and not expired
- Check if you need to verify email first
- Contact HR for new invite link if needed

**Issue: No notifications**
- Check notification settings in Discord
- Verify server notification settings
- Check system notification settings
- Ensure Discord is not muted

**Issue: Cannot hear/be heard in voice channels**
- Check Voice & Video settings
- Test microphone and speakers
- Verify correct input/output devices selected
- Check system audio settings

**Issue: CRM notifications not working**
- Verify Discord ID is added to CRM profile
- Check Discord ID format is correct
- Contact HR if notifications still don't work
