# TASK: Setup Gmail (Work Account)

ID: TASK-009

Category: Onboarding/Communication/Gmail

Tags: [onboarding, email, gmail, communication, workspace]

## Objective

Create and configure a professional Gmail work account with proper security settings, professional signature, and workspace organization. Gmail is used for external communication, account recovery, and integration with other Google services.

## Steps

1. **Create Gmail Account**
   - Visit https://accounts.google.com/signup
   - Click "Create account"
   - Choose "For work" or "For myself" (work account)
   - Enter first and last name
   - Choose username (professional format recommended)
   - Create strong password
   - Add recovery email and phone number

2. **Complete Account Setup**
   - Verify email address
   - Verify phone number if required
   - Complete security questions
   - Accept terms of service
   - Complete profile setup

3. **Configure Security Settings**
   - Go to Google Account settings
   - Enable 2-Step Verification (critical)
   - Set up backup codes
   - Configure recovery options
   - Review security settings

4. **Set Up Professional Profile**
   - Go to Gmail Settings → General
   - Add professional name
   - Upload profile photo (professional headshot)
   - Configure display name
   - Set language and region

5. **Create Professional Email Signature**
   - Go to Settings → General → Signature
   - Create professional signature:
     - Full name
     - Job title
     - Company name
     - Contact information (if appropriate)
     - Professional closing
   - Format signature properly
   - Set as default signature

6. **Configure Privacy Settings**
   - Go to Google Account → Privacy
   - Turn OFF search personalization (if required)
   - Configure data sharing preferences
   - Review privacy settings
   - Adjust as needed for work requirements

7. **Set Up Email Organization**
   - Create labels/folders:
     - Work
     - Important
     - Projects
     - Archive
   - Set up filters for automatic organization
   - Configure inbox categories if needed

8. **Configure Email Settings**
   - Set up email forwarding if needed
   - Configure vacation responder if applicable
   - Set up email delegation if required
   - Configure POP/IMAP if using email client

9. **Link to Chrome Profile**
   - Sign in to Chrome with Gmail account
   - Enable Chrome sync
   - Verify account appears in Chrome
   - Test sync functionality

10. **Set Up Google Workspace Integration**
    - Access Google Drive
    - Access Google Sheets
    - Access Google Calendar
    - Verify all services work with account

11. **Test Email Functionality**
    - Send test email
    - Receive test email
    - Test signature appears correctly
    - Verify security settings work
    - Test 2-Step Verification

## Common Issues and Solutions

**Issue: Cannot create account**
- Verify email format is available
- Check if account already exists
- Try different username variations
- Contact IT if issues persist

**Issue: 2-Step Verification not working**
- Verify phone number is correct
- Check SMS delivery
- Use backup codes if needed
- Try alternative verification method

**Issue: Signature not appearing**
- Check signature is set as default
- Verify signature settings saved
- Clear browser cache
- Test in different email client
