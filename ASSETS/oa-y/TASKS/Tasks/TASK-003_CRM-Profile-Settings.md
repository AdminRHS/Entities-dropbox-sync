# TASK: CRM Profile Settings

ID: TASK-003

Category: Onboarding/CRM/Profile

Tags: [onboarding, crm, profile, settings, employee-setup]

## Objective

Complete your CRM profile settings by filling out all required fields, uploading your profile photo, configuring notification preferences, and setting up your work schedule. A complete profile is essential for time tracking, client presentations, and team collaboration.

## Steps

1. **Access Profile Settings**
   - Log into CRM: https://crm-s.com/
   - Click your name in the top-right corner
   - Select **"Profile"** or **"Settings"**
   - Profile page will open

2. **Fill Required Profile Fields**
   - **Full Name:** Enter your real name (First + Last)
   - **Email:** Enter your work email address
   - **Role:** Verify your role is correctly set (usually pre-set by HR)
   - **Team:** Verify your team assignment (usually pre-set by HR)
   - **Time Zone:** Select your local time zone (e.g., UTC+2 for Kyiv)
   - **Language:** Set interface language (English recommended)

3. **Add Contact Information**
   - **Phone:** Enter phone number (digits only, no +, spaces, or brackets)
   - **Telegram:** Enter Telegram handle with @ symbol (if applicable)
   - **Discord ID:** Add your Discord ID (important for automated reminders)
   - **Personal Email:** Add personal email (optional)
   - **Work Email:** Verify work email is correct

4. **Set Primary Occupation**
   - Navigate to **Primary Occupation** field
   - Select your main job role from dropdown:
     - Lead Generator
     - Developer
     - Designer
     - HR Manager
     - Sales Manager
     - Other (select appropriate option)
   - Verify selection matches your actual role

5. **Add Skills**
   - Navigate to **Skills** field
   - Add all relevant skills for your role
   - Use comma-separated format
   - Include both technical and soft skills
   - Examples:
     - For Developers: "JavaScript, Python, React, Git, Problem Solving"
     - For Lead Generators: "LinkedIn Outreach, CRM Management, Email Communication, Research"
     - For Designers: "UI/UX Design, Figma, Adobe Creative Suite, Branding"

6. **Add Workplace Information**
   - Navigate to **Workplace** field
   - Add company name: "Remote Helpers" or "RemotEmployees"
   - Verify workplace is correctly assigned
   - Add start date if required

7. **Upload Profile Photo**
   - Click **"Change Photo"** or profile icon
   - Select professional headshot from your computer
   - Crop image to square format (face clearly visible)
   - Click **"Save"**

   - Navigate to Profile ‚Üí Work Schedule

   - Select working days (Monday-Friday or your specific schedule)

## Common Issues and Solutions

**Issue: Cannot access profile settings**
- Verify you are logged into CRM
- Clear browser cache: `Ctrl + Shift + Del`
- Try different browser (Chrome recommended)
- Contact HR if access issues persist

**Issue: Profile photo won't upload**
- Check file format (JPG, PNG supported)
- Verify file size (under 5MB recommended)
- Ensure image is square or can be cropped to square
- Try different browser

**Issue: Skills field not saving**
- Verify skills are comma-separated
- Check for special characters that might cause issues
- Try saving one skill at a time
- Contact IT support if problem persists

**Issue: Work timer not working**
- Verify work schedule is set correctly
- Check that Clock In tab is enabled
- Ensure you're in correct time zone
- Contact HR if timer issues continue
