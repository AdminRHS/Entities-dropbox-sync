# TASK: Setup Dropbox

ID: TASK-007

Category: Onboarding/Storage/Dropbox

Tags: [onboarding, storage, file-management, cloud-sync, workspace]

## Objective

Install and configure Dropbox as your primary file storage and workspace system. Dropbox automatically creates daily folders with plan.md, tasks.md, and daily.md files, and syncs your work across all devices.

## Steps

1. **Receive Dropbox Account Credentials**
   - HR will provide Dropbox account credentials
   - May be corporate account or personal account setup
   - Verify you have login credentials before proceeding

2. **Download Dropbox Desktop Application**
   - Visit https://www.dropbox.com/download
   - Download Dropbox for your operating system (Windows/Mac/Linux)
   - Run the installer
   - Follow installation wizard

3. **Sign In to Dropbox**
   - Open Dropbox application
   - Click "Sign In"
   - Enter credentials provided by HR
   - Complete authentication if required

4. **Configure Dropbox Sync Settings**
   - Choose sync location (default is recommended)
   - Select which folders to sync:
     - Sync all folders (recommended for first setup)
     - Or select specific folders to save space
   - Configure selective sync if needed

5. **Verify Company Folder Access**
   - Locate company folder (e.g., "Nov 25", "ELG NOV25", or department folder)
   - Verify you can see your personal folder within company structure
   - Check folder structure matches expected layout

6. **Understand Daily Folder Structure**
   - Dropbox automatically creates daily folders (e.g., "19", "20", "21")
   - Each daily folder contains:
     - `plan.md` - Daily planning file
     - `tasks.md` - Task tracking file
     - `daily.md` - Daily report file
   - These files are automatically populated by Wispr Flow transcripts

7. **Set Up Selective Sync (Optional)**
   - If storage space is limited, configure selective sync
   - Right-click Dropbox icon → Preferences → Sync
   - Choose which folders to sync locally
   - Files with cloud icon (☁️) are online-only
   - Files download when opened

8. **Configure Dropbox Icon Settings**
   - Right-click Dropbox icon in system tray
   - Configure preferences:
     - Startup settings
     - Bandwidth limits if needed
     - Notification preferences
   - Pin Dropbox to taskbar for easy access

9. **Test File Sync**
   - Create a test file in Dropbox folder
   - Verify it syncs to cloud
   - Check Dropbox web interface to confirm sync
   - Test accessing file from another device if available

10. **Set Up Dropbox Web Access**
    - Visit https://www.dropbox.com
    - Sign in with same credentials
    - Verify you can access files via web interface
    - Bookmark Dropbox web interface for quick access

11. **Configure File Sharing (If Needed)**
    - Learn how to share files and folders
    - Understand sharing permissions
    - Practice sharing a test file with team member
    - Verify recipient can access shared content

12. **Verify Integration with VS Code/Cursor**
    - Open VS Code or Cursor
    - File → Open Folder
    - Navigate to Dropbox folder
    - Select your work folder
    - Verify files appear in IDE sidebar

## Common Issues and Solutions

**Issue: Cannot sign in**
- Verify credentials are correct
- Check if account is activated by Finance/IT
- Contact HR if login issues persist

**Issue: Files not syncing**
- Check internet connection
- Verify Dropbox is running
- Check sync status in Dropbox icon
- Restart Dropbox application

**Issue: Not enough disk space**
- Configure selective sync
- Move large files to online-only
- Clean up unnecessary files
- Contact IT if storage issues persist

**Issue: Cannot see company folder**
- Verify you're signed in with correct account
- Check folder permissions
- Contact HR if folder access issues
