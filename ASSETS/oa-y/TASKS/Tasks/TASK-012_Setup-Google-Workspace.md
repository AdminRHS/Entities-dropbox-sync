# TASK: Setup Google Workspace (Drive & Sheets)

ID: TASK-012

Category: Onboarding/Productivity/GoogleWorkspace

Tags: [onboarding, google-workspace, drive, sheets, collaboration, productivity]

## Objective

Set up and configure Google Workspace services including Google Drive for file storage and Google Sheets for data management. These tools are essential for collaboration, document sharing, and data organization across all departments.

## Steps

1. **Access Google Workspace**
   - Sign in to Gmail account (from TASK-009)
   - Click Google Apps icon (9 dots, top-right)
   - Access Google Drive
   - Access Google Sheets
   - Verify all services are accessible

2. **Set Up Google Drive**
   - Go to https://drive.google.com
   - Familiarize yourself with Drive interface
   - Create folder structure:
     - Work
     - Projects
     - Personal
     - Archive
   - Organize folders as needed

3. **Install Google Drive Desktop (Optional)**
   - Visit https://www.google.com/drive/download/
   - Download Google Drive for Desktop
   - Install application
   - Sign in with work Gmail account
   - Choose sync location
   - Configure sync settings

4. **Set Up Google Sheets**
   - Go to https://sheets.google.com
   - Create first spreadsheet
   - Familiarize yourself with Sheets interface
   - Learn basic functions:
     - SUM, AVERAGE, COUNT
     - IF statements
     - VLOOKUP
     - Data validation

5. **Configure Sharing Settings**
   - Learn how to share files/folders
   - Understand permission levels:
     - Viewer
     - Commenter
     - Editor
   - Practice sharing a test file
   - Configure link sharing settings

6. **Set Up Company Folders**
   - Locate company shared folders
   - Request access if needed
   - Organize bookmarks to important folders
   - Understand folder structure
   - Learn naming conventions

7. **Configure Google Docs (If Needed)**
   - Access Google Docs
   - Create test document
   - Learn collaboration features
   - Understand commenting system
   - Practice real-time editing

8. **Set Up Google Calendar Integration**
   - Access Google Calendar
   - Sync with other calendars if needed
   - Set up work calendar
   - Configure event notifications
   - Share calendar with team if needed

9. **Learn File Organization**
   - Understand Drive folder structure
   - Learn file naming conventions
   - Set up file templates
   - Organize files by project/date
   - Use color coding for folders

10. **Configure Offline Access**
    - Enable offline access for Drive
    - Install Google Docs Offline extension
    - Test offline functionality
    - Sync files when online

11. **Set Up Mobile Access**
    - Install Google Drive app on mobile
    - Install Google Sheets app on mobile
    - Sign in with work account
    - Configure mobile sync
    - Test mobile access

## Common Issues and Solutions

**Issue: Cannot access Google Workspace**
- Verify Gmail account is active
- Check if account has Workspace access
- Contact IT if access issues persist

**Issue: Files not syncing**
- Check internet connection
- Verify sync settings
- Restart Google Drive app
- Check storage quota

**Issue: Cannot share files**
- Verify sharing permissions
- Check if link sharing is enabled
- Verify recipient email is correct
- Check file/folder permissions
