## Description
Courses of the Game Academy need cover images and thumbnails that will look nice and stylistically consistent on the main page. This course teaches how to create cover images in the same visual style, visually informative about the course topic, containing one of the mascots for professions and departments, and with a unique non-repetitive environment on the background, appropriate for the mascot and the course topic.
