## Description
This course introduces employees to Discord as a workspace for communication, collaboration, and onboarding. You will learn how to install and use Discord, join the company workspace server, navigate main channels, and work effectively in voice rooms. Step-by-step lessons with examples and screenshots will help you adapt quickly.
