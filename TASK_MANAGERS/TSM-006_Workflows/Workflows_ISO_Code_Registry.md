## Entity Type ISO Codes

| ISO Code | Full Name | Description | Range | Count | Template |
|----------|-----------|-------------|-------|-------|----------|
| **WRF** | Workflow | Reusable workflow patterns that can be applied across projects and milestones | WRF-001 to WRF-020 | 20 | No |

### Code Derivation

**WRF** is derived from:
- **W**o**R**k**F**low
- Uses consonants only (W, R, F)
- Does **NOT** end with **T** (non-template entity)
- Alternative considered: WKF (rejected, less intuitive)
