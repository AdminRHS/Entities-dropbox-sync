## Overview

Project Templates (PRT) are the top-level strategic entities in the task management hierarchy. Each project contains multiple milestones, which in turn contain tasks, steps, and checklist items.

**Hierarchy Position:** **Project (PRT)** → Milestone (MLT) → Task (TST) → Step (STT) → Checklist (CHT)
