## Overview

Task Templates (TST) represent specific deliverables within milestones. Each task contains multiple step templates that describe the procedural execution.

**Hierarchy Position:** Project (PRT) → Milestone (MLT) → **Task (TST)** → Step (STT) → Checklist (CHT)
