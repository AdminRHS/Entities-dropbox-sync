## Usage Workflow

### Step 1: Execute Research
1. Open Perplexity
2. Paste base prompt with tool list
3. Execute research query
4. Review results

### Step 2: Filter Results
1. Verify videos are from past week
2. Check relevance to designers
3. Filter by tutorial quality
4. Prioritize trending content

### Step 3: Compile List
1. Organize by tool
2. Include video titles and links
3. Note view counts/trending status
4. Add brief descriptions

## Expected Output Structure

### Research Results Format

For each tool, provide:
- **Tool Name**
- **Trending Videos** (past week):
  - Video Title
  - Channel Name
  - Upload Date
  - YouTube Link
  - View Count (if available)
  - Brief Description
