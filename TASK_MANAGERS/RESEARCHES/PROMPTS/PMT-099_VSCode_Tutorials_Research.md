# VSCode Tutorials Research - AI Researcher Instructions

**Your Role:** You are a research assistant tasked with finding and analyzing VSCode tutorial videos focused on documentation management, markdown editing, and office productivity.

**Your Task:** Search YouTube and educational platforms to find high-quality VSCode tutorials that teach basic functionality for documentation work, NOT coding or development. Return a comprehensive report with video URLs, analysis, and recommendations.

---

## Research Objective

Find VSCode tutorial videos that teach:
- **Basic keyboard shortcuts** for text editing and navigation
- **Markdown editing** and documentation management
- **Text editing features** (multi-cursor, find/replace, formatting)
- **File management** (opening, saving, organizing files)
- **Using VSCode as a replacement** for Word, Notepad, or other office text editors
- **Productivity features** for non-programmers doing documentation work

**CRITICAL:** You must EXCLUDE videos about:
- Coding, programming, or development workflows
- Debugging, Git advanced features, or development tools
- Content that assumes programming knowledge
- Advanced development features or extension development

---

## Context About the User

The user represents "Remote Helpers," a company that:
- Needs to manage extensive documentation and markdown files
- Wants to use VSCode as a unified tool for documentation instead of Word/Notepad
- Has team members who are NOT developers but need to edit text and markdown files
- Wants to learn basic productivity features and keyboard shortcuts

---

## Your Research Process

### Step 1: Generate Search Queries

Create 8-10 distinct YouTube search queries. Combine:
- "VSCode" or "Visual Studio Code"
- Keywords: "documentation", "markdown", "keyboard shortcuts", "text editing", "office work", "productivity", "for beginners"
- Terms: "tutorial", "guide", "how-to", "tips", "basics"

**Example queries:**
1. "VSCode keyboard shortcuts tutorial for beginners"
2. "VSCode markdown editing guide"
3. "VSCode for documentation management tutorial"
4. "VSCode text editing productivity tips"
5. "VSCode replace Word Notepad office work"
6. "VSCode markdown preview and editing"
7. "VSCode file management documentation"
8. "VSCode basic features tutorial non-programmers"
9. "VSCode markdown extensions setup guide"
10. "VSCode text editing shortcuts productivity"

### Step 2: Search and Filter Videos

**Search YouTube** using your queries. Look for videos that:

**MUST HAVE:**
- Clear screen recordings showing VSCode interface
- Practical examples with real documentation/markdown files
- Step-by-step walkthroughs
- Published within last 90 days (but evergreen content is acceptable)
- Minimum 5 minutes duration
- From credible sources (Microsoft/VSCode official channels, productivity channels, tech educators)

**MUST EXCLUDE:**
- Videos about coding or programming
- Development workflows or debugging tutorials
- Content that assumes programming knowledge
- Videos without screen recordings
- Reaction videos or news without practical demos

### Step 3: Score Each Video (0-100 points)

Rate each video based on:
- **+40 points:** Focuses on documentation management, markdown editing, or office productivity
- **+30 points:** Teaches basic keyboard shortcuts and text editing features
- **+20 points:** Shows practical documentation workflows (not coding)
- **+10 points:** Includes downloadable resources (templates, snippets, configs)

### Step 4: Analyze Each Video

For each video you find, extract:
- Video title and full YouTube URL
- Channel/creator name
- Published date
- Duration
- Main topics covered (with timestamps if available)
- Keyboard shortcuts or features taught
- Extensions or tools mentioned
- Practical workflows demonstrated
- Why it's useful for documentation/office work
