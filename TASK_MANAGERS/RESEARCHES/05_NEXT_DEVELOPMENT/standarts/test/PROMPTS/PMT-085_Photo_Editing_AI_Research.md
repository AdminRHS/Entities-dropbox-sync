# Photo Editing AI Research and Tutorial Compilation

**Purpose**: Research and compile detailed tutorials on photo editing and retouching AI tools through deep research in Gemini.

**Version**: 1.0  
**Date**: 2025-11-21  
**Status**: Active  
**Related Prompts**: PMT-082_Advanced_Prompting_Course_Creation.md

---

## Overview

This prompt enables comprehensive research on photo editing and retouching AI tools, followed by compilation of detailed tutorials based on sourced materials. The process involves multiple iterations to refine and expand the content.

**Primary Use Case**: Creating tutorials on photo editing AI tools for training materials  
**Target Platform**: Gemini (for deep research)  
**Department**: DGN (Design)

---

## Prompt Instructions

### Step 1: Initial Research

```
Please find five most actual AIs for photo retouch and editing, and give me links to latest tutorials on how to use them
```

**Expected Output**: List of 5 AI tools with tutorial links

### Step 2: Compile Tutorials

```
Nice, please compile five tutorials, based on those text-based guides that you sourced in the text. You can use illustrations from them as well
```

**Expected Output**: Compiled tutorials with illustrations

### Step 3: Expand and Detail

```
make them more detailed and more repeating the sources
```

**Expected Output**: Detailed tutorials closely following source materials

---

## Usage Workflow

1. **Execute Step 1**: Run initial research prompt in Gemini
2. **Review Results**: Verify 5 tools identified with tutorial links
3. **Execute Step 2**: Compile tutorials based on research
4. **Review Compilation**: Check tutorial completeness
5. **Execute Step 3**: Expand and add detail
6. **Final Review**: Ensure tutorials are comprehensive and detailed

---

## Expected Output Structure

### For Each AI Tool Tutorial

1. **Tool Introduction**
   - Tool name and purpose
   - Key features
   - Use cases

2. **Tutorial Content**
   - Step-by-step instructions
   - Screenshots/illustrations
   - Tips and best practices
   - Common issues and solutions

3. **Source References**
   - Links to original tutorials
   - Credit to source materials
