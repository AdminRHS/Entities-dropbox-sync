# VIDEO_RESEARCHES Prompt Examples

## Overview

This document contains example prompts for video-related research within the RESEARCHES entity.

## Example Prompts

### RSH-VID-001: Video Tool Comparison Research

```markdown
#Action:Research

Research and compare video processing tools for the following criteria:
1. Transcription accuracy
2. Caption generation quality
3. Integration capabilities
4. Cost efficiency
5. Scalability

Output format: Comparison matrix with scoring rubric.
```

### RSH-VID-002: Influencer Performance Analysis

```markdown
#Action:Analyze

Analyze influencer performance data from REPORTS/Influencer_Tracking:
1. Engagement rate trends
2. Content type performance
3. Platform-specific metrics
4. ROI calculations

Cross-reference with RESEARCHES/DATA for historical patterns.
```

### RSH-VID-003: Video Workflow Optimization

```markdown
#Action:Optimize

Review current video processing workflows in PROMPTS/Video_Processing:
1. Identify bottlenecks
2. Map dependencies
3. Propose automation opportunities
4. Estimate efficiency gains

Link findings to TASK_MANAGERS for implementation tracking.
```

## Template Structure

All VIDEO_RESEARCHES prompts should include:
- **#Action tag** - Define the research action
- **Scope** - What to research
- **Data sources** - Which entities to reference
- **Output format** - Expected deliverable format
- **Links** - Related entities and tasks
