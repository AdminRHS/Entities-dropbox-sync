## Assignment Guidelines

**Before assigning:**
1. Check `ACTIVE_EMPLOYEES_LIST.md` for capacity
2. Review `EMPLOYEE_RESEARCH_HISTORY.md` for past performance
3. Match skills to requirements
4. Consider workload balance
5. Reference `ASSIGNMENT_GUIDELINES.md` for decision tree

**After assigning:**
1. Update this log with assignment details
2. Create task instance in TASK_MANAGERS if needed
3. Notify employee (Discord/Email)
4. Add to `WEEKLY_RESEARCH_PLAN.md`
5. Set reminder for check-in

**During progress:**
1. Weekly check-in with assigned employee
2. Update status in this log
3. Document any blockers
4. Adjust due date if needed

**Upon completion:**
1. Review deliverables
2. Update `RESEARCH_COMPLETION_TRACKER.md`
3. Log in `EMPLOYEE_RESEARCH_HISTORY.md`
4. Move to "Completed Assignments" section
5. Archive task instance
