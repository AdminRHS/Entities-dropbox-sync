## üìö Overview

The Knowledge Library Framework organizes company knowledge using a Library of Congress style classification system adapted for business operations. Every document receives a unique call number for instant retrieval.

---

## üî¢ Call Number Format

**Format:** `[SECTION].[SUBSECTION].[YEAR].[MONTH].[SEQUENCE]`

**Example:** `LG.002.2025.10.024` = Lead Generation, Cluster 2, 2025, October, Document 24

**Components:**
- **SECTION** - Major shelf code (OP, LG, DS, TR, PS, TI, DP, OK)
- **SUBSECTION** - Subsection number (001, 002, 003, etc.)
- **YEAR** - 4-digit year (2025)
- **MONTH** - 2-digit month (01-12)
- **SEQUENCE** - Sequential document number (001, 002, 003, etc.)

## üîç Retrieval Methods

### 1. By Call Number
- **Direct:** `LG.002.2025.10.001`
- **Partial:** `LG.002.*` (all Cluster 2 documents)
- **Year range:** `*.2025.*` (all 2025 documents)

### 2. By Shelf
- **Browse shelf:** `LG/` (all Lead Generation documents)
- **Subsection:** `LG.001/` (Lead Generation Strategy only)

### 3. By Keywords
- Search catalog for: "message templates" ‚Üí Returns `LG.005.*`
- Search catalog for: "onboarding" ‚Üí Returns `TR.001.*`, `PS.004.*`

### 4. By Cross-Reference
- Documents with multiple call numbers
- Related documents listed in metadata
