### 4. **Designer Workflows**

✅ **Designer_Workflows_Overview.md** - `Knowledge_Library/DS/DS.001/Designer_Workflows_Overview.md`

Complete 500+ line workflow guide including:

**Daily Workflow:**
- Morning routine (Clock in via CRM, join Discord voice, check Notion)
- Task assignment process
- Design execution (Drafts → Review → Refinement → Finalization → Upload)
- Golden Rules (Mobile-First ALWAYS, Share working files, Use brand guidelines)
- End of day (Daily report in CRM, clock out)

**5 Task-Specific Workflows:**
1. **Web Banner Design** (2-4 hours)
   - Tools: Figma/Photoshop/Canva
   - 1200x700px standard
   - Steps: Template → Customize → CTA → Review → Export

2. **Logo Design System** (5-10 days)
   - PRIMARY tool: Adobe Illustrator
   - 6 logo types to deliver
   - Complete workflow: Research → Concept → Refinement → Finalization
   - Deliverables: AI, SVG, PNG, EPS, Guidelines PDF

3. **UI/UX Wireframe & Mockup** (1-3 days wireframes, 3-7 days mockups)
   - PRIMARY tool: Figma (all 31+ projects)
   - Mobile-first approach (GOLDEN RULE)
   - Research → Wireframes → High-fidelity → Prototype → Handoff

4. **Rapid Social Media Graphics** (30 min - 1 hour)
   - Tool: Canva for speed
   - When to use: Time-sensitive, quick turnaround
   - Platform templates (Instagram 1080x1080px, LinkedIn 1200x627px)

5. **AI-Enhanced Illustration** (2-5 hours vs 1-2 days manual)
   - Midjourney concept generation → Illustrator vector refinement
   - 50-70% time savings
   - Maintains creative control and brand accuracy

**Tool Selection Guidelines:**
- When to use Figma (PRIMARY for UI/UX)
- When to use Adobe Photoshop (raster graphics, photos)
- When to use Adobe Illustrator (PRIMARY for vectors, logos, icons)
- When to use Canva (speed < 1 hour)
- When to use Midjourney (rapid concepts)
- When to use Gamma AI (presentations < 2 hours)

**Quality Assurance Checklist:**
- Visual quality checks (brand guidelines, typography, accessibility)
- Technical quality (dimensions, file formats, resolution, file size)
- Mobile-first verification (GOLDEN RULE)
- Deliverables checklist
- Communication checklist
