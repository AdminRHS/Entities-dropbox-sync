## Format Selection Matrix

| User Type | Preferred Format | Why |
|-----------|------------------|-----|
| New Employee | Video Tutorial | Visual demonstration, easier to follow |
| Developer | Markdown | Copy-paste examples, version control |
| Manager | Miro Board | High-level overview, visual hierarchy |
| Quick Reference | Markdown | Fast search, ctrl+F functionality |
| Offline User | PDF (from Markdown) | No internet required |
| Mobile User | Video or Social Infographic | Responsive on phone, bite-sized |
| Social Sharing | Infographic/Carousel | Shareable, high engagement |
| Executive | Infographic (single slide) | Quick overview, professional |
